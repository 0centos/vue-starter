# vue-starter

一个基于 Vue3 的前端项目模板。

## 技术栈

- Vue3.5
- Vite5
- Typescript
- Element-Plus
- UnoCSS
- Pinia
- Vue-Router
- unplugin-vue-router
- unplugin-auto-import
- ESLint
- Prettier
- Vitest
- Cypress

## 使用

1. 克隆项目

```bash
git clone https://github.com/0centos/vue-starter.git
```

2. 安装依赖

```bash
cd vue-starter
pnpm install
```

3. 运行项目

```bash
pnpm dev
```

## 项目功能

- 计数器
- 计时器
- Todo List
